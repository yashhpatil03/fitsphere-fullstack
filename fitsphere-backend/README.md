# FitSphere Full Stack Fitness Management System

## Overview

FitSphere is a full-stack fitness tracking platform developed using Spring Boot and React.

Users can:

- Register & Login
- Manage Profile
- Track Workouts
- Track Diet Plans
- Monitor Fitness Progress
- Generate Reports
- Get AI Fitness Guidance using Ollama

---

## Tech Stack

### Frontend

- React.js
- Vite
- Tailwind CSS
- Axios
- React Router

### Backend

- Spring Boot
- Spring Security
- JWT Authentication
- Spring Data JPA
- MySQL

### AI Integration

- Ollama
- Llama3
